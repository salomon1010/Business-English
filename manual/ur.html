1شروعات اور تعارف

ایپ پیشہ ورانہ بولی جانے والی انگریزی کے لیے ایک مکمل 12-ہفتہ تربیتی پروگرام ہے: ہفتے میں 6 دن، روزانہ 25 مرکوز منٹ، ہر ریکارڈنگ پر فوری رائے کے ساتھ۔ سب کچھ خودکار طور پر آپ کے آلے پر محفوظ ہوتا ہے۔

  1. ایپ کھولیں Chrome یا Edge میں (بہترین) — https://app.lomonec.com/۔ Android پر، براؤزر مینو استعمال کریں → ہوم اسکرین میں شامل کریں تاکہ اسے ایک حقیقی ایپ کی طرح انسٹال کریں (یہ آف لائن کام کرتی ہے)۔
  2. 60 سیکنڈ کا خیرمقدم مکمل کریں: آپ کا نام ← آپ کیا کرتے ہیں ← آپ کا سب سے بڑا مقصد ← آپ کا روزانہ مشق کا وقت ← اپنے بنیادی اسکور کے لیے ایک جملہ بلند آواز میں کہیں۔ پوری ایپ آپ کے جوابات کے مطابق خود کو ذاتی بناتی ہے۔
  3. مائیکروفون تک رسائی کی اجازت دیں جب پوچھا جائے — ریکارڈر اور بولنے کی رائے کو اس کی ضرورت ہے۔
  4. آپ 🎯 Shadow ٹیب پر پہنچتے ہیں — ایک ابتدائی ویڈیو منتخب کریں اور اپنا پہلا 30 سیکنڈ کا اسپرنٹ کریں۔ اسی کے ساتھ دن 1 مکمل۔
💡 سنہری اصول: ہر سیشن کو لازماً پیدا کرنا چاہیے ثبوت — ایک ریکارڈنگ، ایک نقل، محفوظ الفاظ، یا اصلاحی نوٹس۔ اگر آپ نے کچھ ریکارڈ یا لکھا نہیں، تو وہ ایک حقیقی سیشن نہیں تھا۔

3ڈیش بورڈ

کوئی فیصلہ نہیں: ڈیش بورڈ آپ کو نام سے خوش آمدید کہتا ہے، آپ کا مقصد اور روزانہ وقت دکھاتا ہے، اور آپ کو بالکل بتاتا ہے کہ آج کا سیشن کیا ہے۔

  1. "جہاں چھوڑا تھا وہیں سے جاری رکھیں" کارڈ پڑھیں اور کلک کریں آج کا 25 منٹ کا سیشن شروع کریں ←.
  2. اپنے اعداد و شمار پر ایک نظر ڈالیں: 🔥 سلسلہ · 🎙️ ریکارڈنگز · 💬 مہارت یافتہ جملے /152 · ✅ سیشن /84 · 🎯 محفوظ شیڈوئنگ کلپس۔
  3. رنگ تین مراحل میں 84 سیشنوں میں سے آپ کا فیصد دکھاتا ہے: بنیاد (ہفتہ 1–4) ← کام کی جگہ روانی (ہفتہ 5–8) ← ایگزیکٹو رابطہ (ہفتہ 9–12)۔

412-ہفتہ سفر

دنسیشن کی قسمآپ جو ثبوت پیدا کرتے ہیں
پیر 🎧تلفظ + کسی مقامی کلپ کی شیڈوئنگنقل + نقل کی ریکارڈنگ
منگل 🎯کاروباری بولنے کی مشقریکارڈنگ v1 ← بہتر v2
بدھ 💬ذخیرہ الفاظ + جملہ بینک10 ذاتی جملے + ریکارڈنگ
جمعرات 🧑‍💼میٹنگ کی نقل / سوال و جوابنقلی میٹنگ کی ریکارڈنگ
جمعہ 🔊تلفظ + پیشکش کی درستگیپہلے/بعد کی ریکارڈنگ
ہفتہ 🎤طویل تقریر (3–5 منٹ)طویل ریکارڈنگ + جائزہ نوٹس
اتوار 📊ہفتہ وار جائزہ اور دوبارہ ترتیباسکور + 3 کامیابیاں / کمزوریاں / اصلاحات

5روزانہ 25 منٹ کا سیشن

ایک صفحہ آپ کی پوری مشق پر مشتمل ہے: ٹائمر، منٹ بہ منٹ سانچہ، براہ راست نقل والا ریکارڈر، آپ کا اسکرپٹ، نوٹس اور خود اسکور۔

  1. ▶ شروع دبائیں 25 منٹ کے ٹائمر پر، اور وقت مقررہ سانچے کے مراحل کی پیروی کریں، ہر ایک پر نشان لگاتے ہوئے۔
  2. "🎬 میری مثال — میں کیا کہنے والا ہوں" لکھیں — اپنے الفاظ میں آپ کا جواب (مثلاً آپ کا خود تعارف)۔ یہی وہ ہدف ہے جس کے مقابلے میں آپ کی ریکارڈنگ کو اسکور کیا جاتا ہے۔
  3. 🎙️ دبائیں اور بولیں۔ جب آپ ریکارڈ کرتے ہیں، ایپ آپ کی نقل کرتی ہے۔ ہدف رکھیں 20–60 سیکنڈ حقیقی گفتگو کا — ایک سیکنڈ کے کلپس کا تجزیہ نہیں کیا جا سکتا۔
  4. روکیں ← مکمل رائے رپورٹ ظاہر ہوتی ہے (سیکشن 7 دیکھیں): لفظ بہ لفظ اسکور، گرامر، ذخیرہ الفاظ، رفتار۔ آپ کے الفاظ خودکار طور پر "🎙️ میری نقل" کو بھی بھر دیتے ہیں۔
  5. دوسرا ٹیک ریکارڈ کریں — 📈 لہریہ آپ کی نئی آواز کو پچھلی پر تہ بہ تہ دکھاتا ہے، رفتار کے فیصلے کے ساتھ، جیسے "4.5 سیکنڈ زیادہ چست — زیادہ پُراعتماد رفتار 📈"۔
  6. اپنی اہم اصلاحات نوٹس میں لکھیں، 1–10 میں خود اسکور دیں، اور ✅ سیشن مکمل کے طور پر نشان زد کریں۔
📌 مثال — ہفتہ 1 منگل، "اپنے بارے میں بتائیں"

میری مثال:

"I currently work as a data engineer. My focus is on the reporting pipeline. I'm responsible for data quality."

آپ اسے ریکارڈ کرتے ہیں، اور رپورٹ دکھاتی ہے: 95% — ایک زرد لفظ (responsible)، ایک گرامر کی گرفت ("I am agree" ← "I agree")، رفتار 132 الفاظ/منٹ ✅۔ آپ 🔊/🎤 سے زرد لفظ کی مشق کرتے ہیں، ٹیک 2 ریکارڈ کرتے ہیں، اور کوشش کی سطر دکھاتی ہے 82% → 95% 📈.

6🎯 شیڈوئنگ اسٹوڈیو — مشق لیب

شیڈوئنگ = کسی مقامی بولنے والے کے ساتھ ساتھ بولنا تاکہ اس کی تال، زور اور سُر کی نقل کر سکیں۔ اسٹوڈیو اسے آسان بنا دیتا ہے: کسی بھی YouTube ویڈیو سے 20–30 سیکنڈ کا کلپ کاٹیں اور شیڈوئنگ کرتے ہوئے اسے لوپ کریں۔

ایک کلپ کاٹیں اور لوپ کریں

  1. ایک ویڈیو منتخب کریں: ایک ✨ ابتدائی کارڈ (Steve Jobs, Simon Sinek, Randy Pausch)، کسی 📺 منتخب چینل کا استعمال کریں، یا کوئی بھی YouTube لنک پیسٹ کریں ← ▶ ویڈیو لوڈ کریں.
  2. چلائیں، پھر ⬇ آغاز نشان زد کریں۔ ایک براہ راست کاؤنٹر ظاہر ہوتا ہے اور سیکنڈ گنتا ہے — 20 سیکنڈ پر یہ سبز ہو جاتا ہے ("اب آپ اختتام نشان زد کر سکتے ہیں ✅")، 30 سیکنڈ پر سنہری رنگ میں دھڑکتا ہے۔
  3. ⬇ اختتام نشان زد کریں ← کلپ اب خودکار طور پر لوپ ہوتا ہے 🔁۔ پلے بیک کے دوران کاؤنٹر آپ کی پوزیشن دکھاتا ہے: 12s / 30s، ہر لوپ پر 0 پر دوبارہ ترتیب پاتے ہوئے۔
  4. کلپ کے اندر گھومیں ⟲ (دوبارہ شروع)، ↺5 / 5↻ (5 سیکنڈ پیچھے/آگے چھلانگ — کلک جمع ہوتے ہیں)، اور رفتار بٹن (1× ← .75× ← .5×) اور 🔁 لوپ آن/آف کے ساتھ۔
  5. ⭐ کلپ محفوظ کریں — محفوظ کلپس ایک ٹیپ سے دوبارہ لوڈ ہوتے ہیں، آپ کے نشانات، نقل اور ریکارڈنگز سلامت رہتے ہوئے۔ ریفریش بھی آپ کا آخری کلپ خودکار طور پر بحال کر دیتا ہے۔

🧍 پوسچر کوچ — بولتے وقت ایگزیکٹو موجودگی

  1. "🧍 پوسچر کوچ" پر کلک کریں ریکارڈر کے نیچے (Shadow یا کسی بھی سیشن صفحے پر) اور کیمرے کی اجازت دیں۔
  2. 3 سیکنڈ سیدھے بیٹھیں — کوچ آپ کی آپ کی سیدھی حالت کے مطابق ترتیب دیتا ہے۔
  3. معمول کے مطابق مشق کریں۔ ایک چھوٹا چمکتا ڈھانچہ اوتار (کبھی آپ کی ویڈیو نہیں) آپ کی عکاسی کرتا ہے: سبز = ایگزیکٹو حالت · زرد = سر جھکا ہوا ہے، فون اوپر کریں · نارنجی چمک = جھکے ہوئے، کندھے پیچھے کریں۔
  4. اندھیرے کمرے میں، اسکرین خودکار طور پر ایک 💡 رنگ لائٹ بن جاتی ہے تاکہ آپ کا چہرہ روشن ہو (بند کرنے کے لیے ایک ٹیپ)۔ اگر ٹریکنگ آپ کو نہ دیکھ سکے، تو یہ نرمی سے رک جاتی ہے اور آپ کو آواز پر توجہ دینے دیتی ہے۔
  5. کوچ روکیں تاکہ آپ کا سیشن نتیجہ ملے — مثلاً "🧍 88% سیدھا" — مشوروں کے ساتھ۔
🔒 صفر کیپچر رازداری: کیمرے کے فریم آپ کے آلے پر تجزیہ ہو کر فوراً ضائع کر دیے جاتے ہیں۔ کوئی ویڈیو یا تصویر کبھی محفوظ یا اپ لوڈ نہیں ہوتی — صرف گمنام عددی پوسچر کوآرڈینیٹس موجود رہتے ہیں، اور وہ کبھی آپ کا آلہ نہیں چھوڑتے۔

شیڈوئنگ کریں اور ریکارڈ کریں

  1. لوپ کی شیڈوئنگ 6–8×: سنیں ← وقفوں میں دہرائیں ← آڈیو کے ساتھ بولیں ← اکیلے بولیں۔
  2. 🎬 کلپ کی نقل لکھیں (جو بولنے والا کہتا ہے) — زور والے الفاظ بڑے حروف میں، وقفے / سے نشان زد کریں۔
  3. 🎙️ دبائیں اور ایک مکمل ٹیک شیڈو کریں۔ روکنے پر: آپ کے الفاظ ظاہر ہوتے ہیں 🎙️ میری نقل میں، اور اسی ریکارڈنگ سے مکمل بولنے کی رائے رپورٹ تیار ہوتی ہے۔
  4. کلپ دہرائیں — ہر ٹیک ایک کوشش ہے: "یہ کلپ · کوشش 3 — 80% ← 95% ← 100% 📈 آپ بہتر ہو رہے ہیں!"

7🗣️ بولنے کی رائے کی وضاحت

ہر ریکارڈنگ کے بعد (یا 🎤 بولنے کی رائے شروع کریں کے ساتھ براہ راست "بلند خوانی" کے بعد)، ایپ اس سے جو آپ نے واقعی کہا ایک مکمل رپورٹ بناتی ہے۔

سیکشنیہ آپ کو کیا بتاتا ہے
اسکور %امریکی-انگریزی اسپیچ انجن نے آپ کے کتنے ہدف الفاظ واضح طور پر پہچانے
لفظ چپسسبز = واضح طور پر پہنچا · زرد = مشق درکار (اس کے بجائے کیا سنا گیا دیکھنے کے لیے ہوور کریں)
🎯 مشق کارڈزہر مسئلہ لفظ کے لیے: 🔈 آپ (آپ کی اپنی ریکارڈنگ سے وہ لمحہ چلاتا ہے) · 🔊 مثال · 🐢 آہستہ · 🎤 اسے کہیں — دوبارہ کوشش کریں (جب آپ درست کر لیتے ہیں تو 100% پر سبز ہو جاتا ہے)
رفتار اور بھرتی الفاظآپ کے فی منٹ الفاظ بمقابلہ مقامی 140–160 زون، اور آپ کی "um/uh" گنتی
کوششیںاس کلپ/سیشن کی اسکور تاریخ 📈/📉 فیصلوں کے ساتھ
📖 گرامر رائےدو کالم — آپ نے کیا اچھا کیا (مددگار افعال، present perfect، جملہ بینک کے جملے استعمال ہوئے!) اور آپ کیا بہتر کر سکتے ہیں ("informations" ← information، "I am agree" ← I agree…)، ہر ایک 🔊 درست-نسخہ-سنیں کے ساتھ
📚 اہم ذخیرہ الفاظآپ کی گفتگو سے نکالے گئے الفاظ CEFR سطحوں کے ساتھ — سیکشن 8 دیکھیں
💡 20+ سیکنڈ بولیں۔ آپ جتنا زیادہ بولیں گے، رپورٹ اتنی ہی بھرپور ہوگی۔ اور یاد رکھیں یہ ایک تخمینہ ہے: ایک "چھوٹا ہوا" لفظ عام طور پر غیر واضح تلفظ کا مطلب ہے، لیکن پس منظر کا شور بھی شمار ہوتا ہے۔

8📚 ذخیرہ الفاظ، مشکل الفاظ اور کوئز

  1. اہم ذخیرہ الفاظ ہر رائے رپورٹ میں ظاہر ہوتا ہے — آپ کی اپنی گفتگو سے الفاظ، سطح بیجز کے ساتھ (A2…C1)۔ کسی لفظ پر ٹیپ کریں اس کی لغت کی تعریف، IPA اور اصلی انسانی آڈیو کے لیے؛ استعمال کریں 🎤 اسے کہیں اپنے آپ کو آزمانے کے لیے؛ 🔖 محفوظ کریں اچھے الفاظ (یا "تمام الفاظ محفوظ کریں")۔
  2. ⬆️ اپنے الفاظ کا معیار بڑھائیں: ایپ آپ کے استعمال کردہ غیر رسمی الفاظ کو پہچانتی ہے اور ایگزیکٹو اپ گریڈ تجویز کرتی ہے — آپ نے "get" کہا ← "obtain" آزمائیں — آڈیو اور محفوظ بٹنوں کے ساتھ۔
  3. 💪 میرے مشکل الفاظ (Shadow کے نیچے): رائے جس لفظ کو بھی نشان زد کرتی ہے وہ یہاں خودکار طور پر جمع ہوتا ہے۔ ہر ایک کی 🔊 / 🐢 / 🎤 سے مشق کریں — جتنی بار آپ نے غلطی کی اتنی بار درست تلفظ کریں اور یہ 🏆 کے ساتھ صاف ہو جاتا ہے۔
  4. 📚 میرا ذخیرہ الفاظ + 🎴 مجھے آزمائیں: آپ کے محفوظ الفاظ ایک ڈیک بن جاتے ہیں۔ کوئز آپ کو ایک لفظ کا مطلب پڑھ کر سناتا ہے — آپ کو لازماً لفظ بلند آواز میں کہنا ہوگا؛ ایپ یادداشت اور تلفظ دونوں کا فیصلہ کرتی ہے۔
📌 مثال — کوئز

"یہ کون سا لفظ ہے؟ — The people with an interest or concern in a business project."

آپ 🎤 دباتے ہیں اور کہتے ہیں "stakeholders" → "🏆 درست اور اچھی طرح تلفظ شدہ!"

9جملے اور 💼 ایگزیکٹو پالش

💼 ایگزیکٹو پالش — ایک رہنما کی طرح کہیں

  1. ٹائپ کریں یا 🎤 بول کر لکھوائیں جو آپ غیر رسمی طور پر کہتے، پھر دبائیں ✨ اسے نکھاریں.
  2. ایگزیکٹو نسخہ پڑھیں — بھرتی الفاظ ہٹائے، ہچکچاہٹ کٹی ("I think", "maybe", "sorry about that")، گرامر درست، کمزور الفاظ بہتر — ہر تبدیلی درج تاکہ آپ طرز سیکھیں۔
  3. 🔊 سنیں، 🐢 آہستہ کریں، 📋 کاپی کریں — یا دبائیں 🤖 AI کوچ پرامپٹ تاکہ ChatGPT، Claude یا Gemini کے لیے ایک تیار کوچنگ پرامپٹ (اندر آپ کے جملے کے ساتھ) کاپی کریں۔
📌 مثال

"um I think maybe we need to fix the problem with the informations"

→ "I recommend we resolve the challenge with the information."

جملہ بینک

ہر ہفتے 10–13 جملے اور محاورے، موضوع سے ہم آہنگ۔ ہر ایک کے لیے: اسے 5× کہیں، 🎙️ ریکارڈ کریں اپنا جملہ، اور مہارت یافتہ نشان زد کریں جب یہ قدرتی طور پر آپ کا بن جائے۔ اتوار کا کام: اگلے ہفتے حقیقی میٹنگوں میں استعمال کے لیے 5 منتخب کریں۔

10جائزے اور 📈 پیش رفت اسٹوڈیو

  1. 📖 اس ہفتے آپ کی کہانی — آپ کی مشق کا ایک خودکار تحریری خلاصہ: دن، بولے گئے الفاظ، اسکور کا رجحان، نیا ذخیرہ الفاظ، صاف کرنے کے مشکل الفاظ۔
  2. چار چارٹ — 🎯 تلفظ اسکور سفر · 📚 ذخیرہ الفاظ میں اضافہ · ⚡ بولنے کی رفتار گیج (وارمنگ اپ / بالکل ٹھیک / تیز) · 🗓️ روزانہ بولے گئے الفاظ۔ سب خودکار طور پر ماپا جاتا ہے۔
  3. ہر اتوار: 2 ریکارڈنگز سنیں، پانچ 1–10 سلائیڈر حرکت دیں (وضاحت، روانی، تلفظ، کاروباری فطری پن، اعتماد)، 3 کامیابیاں / 3 کمزوریاں / 3 اصلاحات لکھیں، اور جائزہ مکمل کریں۔
  4. ہفتہ 4، 8، 12: ماہانہ ڈیش بورڈ بھریں — بڑا چیک پوائنٹ جو اگلا مرحلہ طے کرتا ہے۔

11👤 اکاؤنٹ، بیک اپ اور رازداری

  1. پروفائل: کسی بھی وقت اپنا نام، کردار، مقصد اور روزانہ وقت میں ترمیم کریں — پوری ایپ دوبارہ ذاتی بن جاتی ہے۔
  2. پلان: مفت پلان میں سب کچھ شامل ہے؛ پریمیم (AI کوچ، کلاؤڈ اکاؤنٹس، ٹیم ڈیش بورڈ) بعد میں آ رہا ہے۔
  3. بیک اپ: کلک کریں JSON برآمد کریں ہفتہ وار اور فائل محفوظ رکھیں؛ درآمد اسے کسی بھی آلے پر بحال کرتا ہے۔ اختیاری: پیش رفت اور ریکارڈنگز کی خودکار کلاؤڈ ہم آہنگی کے لیے ایک نجی GitHub ریپو منسلک کریں۔
  4. رازداری: سب کچھ — پیش رفت، ریکارڈنگز، نقلیں — آپ کے آلے پر رہتا ہے۔ کچھ بھی اپ لوڈ نہیں ہوتا جب تک آپ GitHub ہم آہنگی مرتب نہ کریں۔
⛔ "سب کچھ دوبارہ ترتیب دیں" اس آلے پر تمام پیش رفت اور ریکارڈنگز مستقل طور پر مٹا دیتا ہے۔ پہلے ایک بیک اپ برآمد کریں۔

13عمومی سوالات اور مسائل کا حل

مائیکروفون / بولنے کی رائے کام نہیں کرتی

استعمال کریں Chrome یا Edge — تقریر کی شناخت کو ان کے انجن کی ضرورت ہے (Safari جزوی طور پر کام کرتا ہے؛ Firefox نہیں)۔ ایڈریس بار میں 🔒 آئیکن پر کلک کریں ← مائیکروفون ← اجازت دیں ← دوبارہ لوڈ کریں۔ macOS پر System Settings ← Privacy ← Microphone بھی چیک کریں۔

میں نے ریفریش کیا اور میرا کلپ/ریکارڈنگز غائب لگیں

درست ہو گیا — Shadow ٹیب اب ریفریش کے بعد آپ کا آخری کلپ، نشانات، نقل اور ریکارڈنگز خودکار طور پر بحال کرتا ہے۔ ریکارڈنگز ہمیشہ ریفریش میں محفوظ رہتی ہیں؛ وہ آپ کے براؤزر کے ڈیٹابیس میں رہتی ہیں۔

میری پیش رفت غائب ہو گئی

شاید آپ نے کوئی مختلف براؤزر یا نجی ونڈو کھولی۔ اپنے معمول کے براؤزر پر واپس جائیں، یا اپنا JSON بیک اپ بحال کریں (👤 اکاؤنٹ)۔ اسی لیے ہفتہ وار برآمد کی عادت اہم ہے۔

میری ریکارڈنگ 0:00 دکھاتی ہے / تجزیہ نہیں کیا جا سکتا

یہ بہت مختصر تھی۔ کم از کم 20 سیکنڈ بولیں — آپ ہر جملہ کہنے کے ساتھ رائے کا معیار بڑھتا ہے۔ (نئی ریکارڈنگز آپ کے پلے دبانے تک "0:00" دکھا سکتی ہیں — یہ براؤزر کی ایک خصوصیت ہے، ڈیٹا کا نقصان نہیں۔)

کیا میں اسے ایک ایپ کے طور پر انسٹال کر سکتا ہوں؟

ہاں — Android Chrome پر: مینو ⋮ → ہوم اسکرین میں شامل کریں / ایپ انسٹال کریں۔ یہ نچلی نیویگیشن کے ساتھ فل اسکرین کھلتی ہے اور آف لائن کام کرتی ہے۔

کیا پوسچر کوچ مجھے ریکارڈ کرتا ہے؟

نہیں۔ کیمرہ فیڈ آپ کے آلے پر فریم بہ فریم تجزیہ ہوتی ہے اور ہر فریم فوراً ضائع کر دیا جاتا ہے۔ کوئی ویڈیو یا تصویر کبھی محفوظ یا منتقل نہیں ہوتی — کوچ کے چلنے کے دوران صرف گمنام پوسچر کوآرڈینیٹس (نمبر) موجود رہتے ہیں۔ کوچ بند کرنے سے کیمرہ مکمل طور پر آزاد ہو جاتا ہے۔

کیا میری آواز کسی سرور پر جاتی ہے؟

ریکارڈنگز آپ کے آلے پر رہتی ہیں۔ نقل کے لیے استعمال ہونے والا اسپیچ انجن آپ کے براؤزر کا اپنا ہے (Chrome اپنی اسپیچ سروس کے حصے کے طور پر Google کے سرورز پر آڈیو پروسیس کر سکتا ہے)؛ لغت کی تعریفیں گمنام طور پر لائی جاتی ہیں۔ جب تک آپ GitHub ہم آہنگی فعال نہ کریں، اور کچھ آپ کا آلہ نہیں چھوڑتا۔
