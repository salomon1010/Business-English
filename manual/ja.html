1はじめに・初期設定

このアプリは、プロフェッショナルな英語スピーキングのための完全な12週間トレーニングプログラムです： 週6日、1日25分の集中、録音ごとに即時フィードバックがあります。すべては自動的にあなたの端末に保存されます。

  1. アプリを開く ChromeまたはEdgeで（推奨）— https://app.lomonec.com/。Androidではブラウザのメニュー → ホーム画面に追加 を使って本物のアプリのようにインストールできます（オフラインでも動作します）。
  2. 60秒のウェルカムを完了しましょう： 名前 → 職業 → 最優先の目標 → 毎日の練習時間帯 → 基準スコアのために一文を声に出す。アプリ全体があなたの回答に合わせて最適化されます。
  3. マイクへのアクセスを許可 してください — レコーダーとスピーキング・フィードバックに必要です。
  4. 🎯 Shadowタブに到着します — スターター動画を選んで、最初の30秒スプリントをやりましょう。これで1日目は完了です。
💡 黄金ルール： すべてのセッションは 証拠 を生み出さなければなりません — 録音、書き起こし、保存した単語、または修正メモ。何も録音も記入もしていなければ、本当のセッションではありません。

迷う必要なし：ダッシュボードは名前であなたを迎え、目標と毎日の時間帯を示し、今日のセッションが何かを正確に教えてくれます。

5毎日25分のセッション

1ページにトレーニングのすべてが収まっています：タイマー、分刻みのテンプレート、リアルタイム書き起こし付きレコーダー、あなたの台本、メモ、自己採点。

  1. ▶ 開始 を押して 25分タイマーを動かし、時間管理されたテンプレートの手順に従い、それぞれにチェックを入れていきます。
  2. 「🎬 私の例 — 言うつもりの内容」を書きます — 自分の言葉での回答（例：自己紹介）。これが、あなたの録音が採点される際の基準になります。
  3. 🎙️ を押して話します。 録音中、アプリがあなたの発話を書き起こします。目安は 20〜60秒 の実際の発話です — 1秒のクリップは分析できません。
  4. 停止 → 完全なフィードバックレポートが表示されます （第7節を参照）：単語ごとのスコア、文法、語彙、話す速さ。あなたの発話は「🎙️ 私の書き起こし」にも自動で入ります。
  5. 2回目を録音する — 📈 波形が新しい声を前の声に重ねて表示し、次のような速さの判定が出ます 「4.5秒短縮 — より自信のある速さ 📈」。
  6. 最も重要な修正をメモに書き、1〜10で自己採点し、✅ セッションを完了としてマークします。
📌 例 — 第1週火曜、「自己紹介をしてください」

私の例：

"I currently work as a data engineer. My focus is on the reporting pipeline. I'm responsible for data quality."

録音すると、レポートに表示されます：95% — 黄色の単語が1つ（responsible）、文法の指摘（「I am agree」→「I agree」）、速さ132語/分 ✅。🔊/🎤 で黄色の単語を練習し、2回目を録音すると、挑戦の行に次のように表示されます 82% → 95% 📈.

6🎯 シャドーイング・スタジオ — 練習ラボ

シャドーイング = ネイティブ話者と 一緒に話し ネイティブ話者と一緒に話し、そのリズム・強勢・抑揚を真似ることです。スタジオならそれが簡単に：内蔵ライブラリからクリップを選ぶ（書き起こし付き）か、任意の YouTube リンクを貼り付け、20〜30 秒の区間をマークして、シャドーイングしながらループ再生します。

クリップを切り出してループ再生する

  1. 動画を選ぶ： 下の 🎬 ライブラリからクリップを選ぶ（Jobs、MLK、Michelle Obama、TED トークなど多数——それぞれ「字幕あり」バッジが表示されます）か、任意の YouTube リンクを貼り付け → ▶ 動画を読み込む.
  2. 再生し、次に ⬇ 開始をマーク。 ライブカウンターが表示され秒数を数えます — 20秒で緑になり（「今、終わりをマークできます ✅」）、30秒で金色に脈打ちます。
  3. ⬇ 終わりをマーク → クリップは 自動的にループ再生されます 🔁。再生中、カウンターは現在位置を示します： 12s / 30s、ループごとに0にリセットされます。
  4. 書き起こしは自動で入力されます。 ライブラリのクリップでは、終了をマークした瞬間に、ちょうどその区間の字幕が「🎬 クリップの書き起こしとメモ」欄に取り込まれ、すでに強勢が付いています（大文字＝強勢部分）。マークを変えたあとは「📄 書き起こしを取得」で再取得、「✕ クリア」で新しいクリップを開始できます。
  5. クリップ内を移動する ⟲（最初から）、↺5 / 5↻（5秒戻る/進む — クリックは累積）、 速度ボタン （1× → .75× → .5×）、🔁 ループのオン/オフ を使います。
  6. ⭐ クリップを保存 — 保存したクリップはワンタップで再読み込みされ、マーク、書き起こし、録音はそのまま残ります。更新しても、最後のクリップが自動的に復元されます。

🧍 姿勢コーチ — 話しながらエグゼクティブな存在感を

  1. 「🧍 姿勢コーチ」をクリック （レコーダーの下、Shadowまたは任意のセッションページ）し、カメラを許可します。
  2. 3秒間まっすぐ座る — コーチは あなたの まっすぐな姿勢に合わせて調整します。
  3. 普段どおり練習します。 小さく光るスケルトンのアバター（あなたの映像ではありません）があなたを映します： 緑 = エグゼクティブな姿勢 · 黄 = 頭が傾いています、スマホを上げてください · オレンジの光 = 猫背です、肩を後ろに。
  4. 暗い部屋では、画面が自動的に 💡 リングライト になり、顔を照らします（ワンタップでオフ）。トラッキングがあなたを認識できない場合は、自然に一時停止し、声に集中できるようにします。
  5. コーチを停止 すると、セッション結果が得られます — 例： 「🧍 まっすぐ88%」 — アドバイス付き。
🔒 ゼロキャプチャのプライバシー： カメラのフレームはあなたの端末上で分析され、即座に破棄されます。動画や写真が保存・アップロードされることは一切ありません — 存在するのは匿名の数値の姿勢座標だけで、それらが端末を離れることは決してありません。

シャドーイングして録音する

  1. ループを6〜8回シャドーイング： 聞く → 間で繰り返す → 音声と一緒に話す → 一人で話す。
  2. 🎬 クリップの書き起こしを確認します。 ライブラリのクリップでは、あなたのマークからすでに入力され強勢が付いています。貼り付けた動画では、話者が言う内容を入力して「✨ 強勢を付ける」をタップします。強勢部分は大文字で表示され、ポーズは / で示されます。
  3. 🎙️ を押して、1回分を通してシャドーイングします。 停止すると、あなたの発話が 🎙️ 私の書き起こしに表示され、その録音からスピーキング・フィードバックの完全なレポートが生成されます。
  4. クリップを繰り返す — 各回が1つの挑戦です： 「このクリップ · 挑戦3 — 80% → 95% → 100% 📈 上達しています！」

7🗣️ スピーキング・フィードバックの解説

録音のたびに（または 🎤 スピーキング・フィードバック開始 でのライブ「音読」の後に）、アプリは実際に話した内容から完全なレポートを作成します。

項目何を教えてくれるか
スコア %アメリカ英語の音声エンジンが、あなたのターゲット単語をいくつ明瞭に認識したか
単語チップ緑 = 明瞭に伝わった · 黄 = 練習が必要（ホバーすると代わりに何と聞こえたか表示）
🎯 練習カード問題のある各単語について： 🔈 あなた （あなた自身の録音のその瞬間を再生）· 🔊 お手本 · 🐢 ゆっくり · 🎤 言ってみる — もう一度 （うまく言えて100%になると緑に変わります）
速さとつなぎ言葉あなたの毎分単語数とネイティブの140〜160ゾーンの比較、および「um/uh」の回数
挑戦このクリップ/セッションのスコア履歴と 📈/📉 の判定
📖 文法フィードバック2つの列 — うまくできたこと （法助動詞、present perfect、フレーズバンクのフレーズを使えた！）と 改善できること （「informations」→ information、「I am agree」→ I agree…）、それぞれに 🔊 正しい版を聞く 付き
📚 重要語彙あなたの発話から抽出された単語とCEFRレベル — 第8節を参照
💡 20秒以上話しましょう。 話せば話すほどレポートは充実します。そしてこれは近似であることを覚えておいてください：「聞き取れなかった」単語はたいてい発音が不明瞭という意味ですが、背景の雑音も影響します。

8📚 語彙・苦手な単語・クイズ

  1. 重要語彙 はすべてのフィードバックレポートに表示されます — あなた自身の発話 から抽出された単語で、レベルバッジ付き（A2…C1）。単語をタップすると、その 辞書の定義、IPA、そして本物の人の音声が表示されます； 🎤 言ってみる で自分をテストし； 🔖 保存 して良いものを残します（または「すべての単語を保存」）。
  2. ⬆️ 単語のレベルを上げる： アプリはあなたが使ったくだけた単語を見つけ、より洗練された言い換えを提案します — あなたは「get」と言った → 「obtain」を試してみて — 音声と保存ボタン付き。
  3. 💪 私の苦手な単語 （Shadowの一番下）：フィードバックが指摘した単語がすべてここに自動で集まります。🔊 / 🐢 / 🎤 で一つずつ練習 — 間違えた回数だけ正しく発音すれば、🏆 とともにクリアされます。
  4. 📚 私の語彙 + 🎴 クイズして： 保存した単語がデッキになります。クイズは単語の 意味 を読み上げ — あなたは その単語を声に出して言う必要があります。アプリは記憶と発音の両方を判定します。
📌 例 — クイズ

「これは何の単語でしょう？ — The people with an interest or concern in a business project."

あなたは 🎤 を押して言います 「stakeholders」 → 「🏆 正解、しかも発音も上手！」

9フレーズと 💼 エグゼクティブ・ポリッシュ

💼 エグゼクティブ・ポリッシュ — リーダーのように言う

  1. 入力するか 🎤 で口述 して、くだけた言い方を入れ、次に押します ✨ 磨く.
  2. 洗練された版を読む — つなぎ言葉を除去、ぼかし表現を削除（「I think」「maybe」「sorry about that」）、文法を修正、弱い語を格上げ — すべての変更が一覧表示され、その型を学べます。
  3. 🔊 聞く、🐢 ゆっくりにする、📋 コピーする — または押します 🤖 AIコーチ用プロンプト を押すと、ChatGPT・Claude・Gemini 用の（あなたの文が入った）出来合いのコーチング・プロンプトをコピーできます。
📌 例

"um I think maybe we need to fix the problem with the informations"

→ "I recommend we resolve the challenge with the information."

10振り返りと 📈 進捗スタジオ

  1. 📖 今週のあなたの物語 — あなたの練習の自動生成サマリー：日数、話した語数、スコアの傾向、新しい語彙、クリアすべき苦手な単語。
  2. 4つのグラフ — 🎯 発音スコアの歩み · 📚 語彙の成長 · ⚡ 話す速さのゲージ（ウォームアップ / ちょうどよい / 速い）· 🗓️ 1日に話した語数。すべて自動で測定。
  3. 毎週日曜： 2つの録音を聞き、5つの1〜10スライダー（明瞭さ、流暢さ、発音、ビジネスでの自然さ、自信）を動かし、3つの成果 / 3つの弱点 / 3つの修正を書き、振り返りを完了します。
  4. 第4・8・12週： 月次ダッシュボードを記入します — 次のフェーズを決める大きなチェックポイントです。

11👤 アカウント、バックアップ、プライバシー

  1. プロフィール： 名前、役割、目標、毎日の時間帯をいつでも編集できます — アプリ全体が再度最適化されます。
  2. プラン： 無料プランにはすべて含まれます。プレミアム（AIコーチ、クラウドアカウント、チームダッシュボード）は後日提供予定です。
  3. バックアップ： クリックして JSONをエクスポート を毎週行い、ファイルを安全に保管します； インポート はどの端末でも復元します。任意：プライベートなGitHubリポジトリを接続すると、進捗と録音を自動でクラウド同期できます。
  4. プライバシー： すべて — 進捗、録音、書き起こし — はあなたの端末上にあります。次の場合を除き、何もアップロードされません： あなた がGitHub同期を設定した場合。
⛔ 「すべてリセット」 は、この端末上のすべての進捗と録音を完全に消去します。先にバックアップをエクスポートしてください。

13よくある質問とトラブル対処

マイク / スピーキング・フィードバックが動かない

使ってください ChromeまたはEdge を — 音声認識にはそれらのエンジンが必要です（Safariは一部動作、Firefoxは非対応）。アドレスバーの 🔒 アイコンをクリック → マイク → 許可 → 再読み込み。macOSでは システム設定 → プライバシー → マイク も確認してください。

更新したら、クリップ/録音が消えたようです

修正済み — Shadowタブは更新後、最後のクリップ、マーク、書き起こし、録音を自動で復元するようになりました。録音は更新後も常に残ります。ブラウザのデータベース内にあります。

進捗が消えました

おそらく別のブラウザかプライベートウィンドウを開いています。いつものブラウザに戻るか、JSONバックアップを復元してください（👤 アカウント）。だからこそ、毎週エクスポートする習慣が大切です。

録音が0:00と表示される / 分析できない

短すぎました。少なくとも 20秒 話してください — 一文話すごとにフィードバックの質が上がります。（録りたての録音は、再生を押すまで「0:00」と表示されることがあります — これはブラウザの癖で、データの損失ではありません。）

アプリとしてインストールできますか？

はい — Android Chromeでは：メニュー ⋮ → ホーム画面に追加 / アプリをインストール。下部ナビゲーション付きの全画面で開き、オフラインでも動作します。

姿勢コーチは私を録画しますか？

いいえ。カメラ映像はあなたの端末上でフレームごとに分析され、各フレームは即座に破棄されます。動画や画像が保存・送信されることは一切ありません — コーチの動作中に存在するのは匿名の姿勢座標（数字）だけです。コーチをオフにすると、カメラは完全に解放されます。

私の声はサーバーに送られますか？

録音はあなたの端末に残ります。書き起こしに使う音声エンジンはブラウザ自身のものです（Chromeはその音声サービスの一部として、Googleのサーバーで音声を処理する場合があります）。辞書の定義は匿名で取得されます。GitHub同期を有効にしない限り、他に端末を離れるものはありません。

どの順番で週を進めるべきですか？

順番どおり、1 → 12です。各フェーズは積み重なっており、第4・8・12週は月次ダッシュボードで締めくくられます。
