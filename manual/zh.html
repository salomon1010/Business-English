1开始使用与初次引导

本应用是一套完整的 12 周专业英语口语训练计划： 每周 6 天，每天专注 25 分钟，每次录音都有即时反馈。一切都会自动保存在你的设备上。

  1. 打开应用 使用 Chrome 或 Edge（效果最佳）—— https://app.lomonec.com/。在 Android 上，使用浏览器菜单 → 添加到主屏幕 ，即可像真正的应用一样安装它（可离线使用）。
  2. 完成 60 秒的欢迎流程： 你的名字 → 你的职业 → 你的首要目标 → 你每天练习的时段 → 大声说一句话得到你的初始分。整个应用会围绕你的回答进行个性化。
  3. 允许访问麦克风 ——当被询问时，录音器和口语反馈都需要它。
  4. 你会进入 🎯 Shadow 标签页 ——选一个入门视频，完成你的第一次 30 秒冲刺。这样第 1 天就完成了。
💡 黄金法则： 每次练习都必须产生 凭证 ——一段录音、一份文字稿、保存的单词或纠错笔记。如果你没有录音也没有写任何东西，那就不算一次真正的练习。

3仪表盘

无需做决定：仪表盘会用你的名字问候你，显示你的目标和每日时段，并准确告诉你今天的练习是什么。

  1. 阅读“从上次中断处继续”卡片 并点击 开始今天的 25 分钟练习 →.
  2. 看一眼你的统计数据： 🔥 连续天数 · 🎙️ 录音 · 💬 已掌握短语 /152 · ✅ 练习 /84 · 🎯 已保存的跟读片段。
  3. 进度环 显示你在三个阶段中完成 84 次练习的百分比：基础（第 1–4 周）→ 职场流利度（第 5–8 周）→ 高管沟通（第 9–12 周）。

5每日 25 分钟练习

一个页面涵盖你的全部训练：计时器、逐分钟模板、带实时转写的录音器、你的讲稿、笔记和自评分。

  1. 按下 ▶ 开始 启动 25 分钟计时器，并按计时模板步骤逐一打勾完成。
  2. 写下“🎬 我的示例——我打算说的话” ——用你自己的话作答（例如你的自我介绍）。这是你的录音被评分时对照的目标。
  3. 按下 🎙️ 并开口说。 在你录音时，应用会为你转写。目标是 20–60 秒 的真实口语——一秒的片段无法分析。
  4. 停止 → 完整的反馈报告出现 （见第 7 节）：逐词评分、语法、词汇、语速。你的话也会自动填入“🎙️ 我的文字稿”。
  5. 录制第二遍 ——📈 波形会把你的新声音叠加在上一遍之上，并给出语速判定，例如 “缩短了 4.5 秒——语速更自信 📈”。
  6. 在笔记中写下你最重要的纠正，自评 1–10 分，然后 ✅ 标记练习完成。
📌 示例——第 1 周周二，“介绍一下你自己”

我的示例：

"I currently work as a data engineer. My focus is on the reporting pipeline. I'm responsible for data quality."

你录下它，报告显示：95%——一个黄色单词（responsible），一处语法纠正（“I am agree” → “I agree”），语速 132 词/分 ✅。你用 🔊/🎤 练习那个黄色单词，录制第 2 遍，尝试记录显示 82% → 95% 📈.

6🎯 跟读工作室——练习实验室

跟读 = 与 同步说话 母语者一起说，以复制他们的节奏、重音和语调。工作室让这一切毫不费力：从内置库中选一个片段（已含文字稿），或粘贴任意 YouTube 链接，标记一个 20–30 秒的片段，边跟读边循环播放。

剪辑并循环播放片段

  1. 选一个视频： 从下方的 🎬 库中选择一个片段（Jobs、MLK、Michelle Obama、TED 演讲等等——每个都显示“字幕已就绪”徽章），或粘贴任意 YouTube 链接 → ▶ 加载视频.
  2. 播放，然后 ⬇ 标记开始。 会出现一个实时计数器并计算秒数——到 20 秒时变绿（“现在可以标记结束了 ✅”），到 30 秒时闪烁金色。
  3. ⬇ 标记结束 → 该片段现在会 自动循环播放 🔁。播放期间计数器显示你的位置： 12s / 30s，每次循环归零。
  4. 文字稿会自动为你填入。 对于库中的片段，你一标记结束，应用就会把恰好那一段的字幕拉入“🎬 片段文字稿与笔记”框中，并已标注重音（大写＝重读部分）。更改标记后，点击“📄 获取文字稿”重新提取，或点击“✕ 清除”开始一个新片段。
  5. 在片段内导航 用 ⟲（重新开始）、↺5 / 5↻（后退/前进 5 秒——点击可累加）、 速度按钮 （1× → .75× → .5×）以及 🔁 循环开/关。
  6. ⭐ 保存片段 ——已保存的片段轻点一下即可重新加载，你的标记、文字稿和录音都完好无损。刷新也会自动恢复你上一个片段。

🧍 姿态教练——说话时的高管气场

  1. 点击“🧍 姿态教练” （在录音器下方，Shadow 或任意练习页面）并允许摄像头。
  2. 端坐 3 秒 ——教练会校准到 你的 端正姿态。
  3. 正常练习。 一个发光的小骨架头像（绝不是你的视频）会镜像你的动作： 绿色 = 高管姿态 · 黄色 = 头部倾斜，请抬高手机 · 橙色光晕 = 弯腰驼背，请挺起肩膀。
  4. 在昏暗的房间里，屏幕会自动变成一盏 💡 环形灯 来照亮你的脸（轻点一下即可关闭）。如果追踪看不到你，它会优雅地暂停，让你专注于声音。
  5. 停止教练 即可获得本次练习结果——例如 “🧍 端正 88%” ——并附有建议。
🔒 零采集隐私： 摄像头画面在你的设备上分析后立即丢弃。绝不会存储或上传任何视频或照片——只存在匿名的数字姿态坐标，且它们绝不会离开你的设备。

跟读并录音

  1. 跟读循环 6–8 次： 听 → 在停顿处重复 → 跟着音频说 → 独立说。
  2. 检查 🎬 片段文字稿。 对于库中的片段，它已根据你的标记填入并标注重音；对于粘贴的视频，输入说话者所说的话并点击“✨ 标注重音”。重读部分以大写显示，停顿用 / 标记。
  3. 按下 🎙️ 完整跟读一遍。 停止时：你的话会出现在 🎙️ 我的文字稿，完整的口语反馈报告会根据那次录音生成。
  4. 重复该片段 ——每一遍都是一次尝试： “此片段 · 第 3 次尝试——80% → 95% → 100% 📈 你在进步！”

7🗣️ 口语反馈详解

每次录音后（或用 🎤 开始口语反馈进行实时“朗读”后），应用会根据你实际说的话生成完整报告。

板块它告诉你什么
分数 %美式英语语音引擎清晰识别出了你多少个目标单词
单词标签绿色 = 说得清晰 · 黄色 = 需要练习（悬停可查看实际听到的内容）
🎯 练习卡片针对每个有问题的单词： 🔈 你 （播放你自己录音中的那个瞬间）· 🔊 示例 · 🐢 慢速 · 🎤 说出来——再试一次 （当你达到 100% 说对时会变绿）
语速与口头禅你的每分钟词数对比母语者的 140–160 区间，以及你的“um/uh”次数
尝试此片段/练习的分数历史，附 📈/📉 判定
📖 语法反馈两栏—— 你做得好的地方 （情态动词、present perfect、用上了短语库里的短语！）和 你可以改进的地方 （“informations” → information，“I am agree” → I agree……），每条都带 🔊 听听正确版本
📚 关键词汇从你的口语中提取的单词，附 CEFR 等级——见第 8 节
💡 说满 20 秒以上。 你说得越多，报告越丰富。也请记住这只是一个近似值：一个“漏掉”的单词通常意味着发音不清，但背景噪音也会有影响。

8📚 词汇、难词与测验

  1. 关键词汇 出现在每份反馈报告中——来自 你自己口语 的单词，附等级徽章（A2…C1）。轻点一个单词查看它的 词典释义、国际音标和真人发音；用 🎤 说出来 来测试自己； 🔖 保存 好的单词（或“保存全部单词”）。
  2. ⬆️ 提升你的用词： 应用会识别你用过的随意词汇，并建议更高管的升级版—— 你说了“get” → 试试“obtain” ——附音频和保存按钮。
  3. 💪 我的难词 （Shadow 底部）：反馈标记的每个单词都会自动汇集到这里。用 🔊 / 🐢 / 🎤 练习每一个——正确发音的次数达到你出错的次数，它就会带着 🏆 被清除。
  4. 📚 我的词汇 + 🎴 来测我： 你保存的单词会变成一副卡片。测验会读出一个单词的 含义 ——你必须 大声说出这个单词；应用会同时评判你的记忆和发音。
📌 示例——测验

“这是哪个单词？—— The people with an interest or concern in a business project."

你按下 🎤 并说 “stakeholders” → “🏆 答对了而且发音很好！”

9短语与 💼 高管润色

💼 高管润色——像领导者一样表达

  1. 输入或 🎤 口述 你随口会说的话，然后按下 ✨ 润色它.
  2. 阅读高管版本 ——去掉口头禅、删去含糊措辞（“I think”“maybe”“sorry about that”）、修正语法、升级弱词——并列出每一处改动，让你学会其中的规律。
  3. 🔊 听它、🐢 放慢它、📋 复制它 ——或按下 🤖 AI 教练提示词 以复制一个现成的教练提示词（内含你的句子），可用于 ChatGPT、Claude 或 Gemini。
📌 示例

"um I think maybe we need to fix the problem with the informations"

→ "I recommend we resolve the challenge with the information."

短语库

每周 10–13 条短语和习语，与主题匹配。每条：说 5 遍， 🎙️ 录下 你自己的句子，并在它 标记为已掌握 自然成为你的一部分时如此标记。周日的任务：挑 5 条下周在真实会议中使用。

10回顾与 📈 进度工作室

  1. 📖 你本周的故事 ——自动撰写的练习总结：天数、说过的词数、分数趋势、新词汇、待清除的难词。
  2. 四张图表 ——🎯 发音分数历程 · 📚 词汇增长 · ⚡ 语速仪表（热身中 / 恰到好处 / 偏快）· 🗓️ 每日说过的词数。全部自动测量。
  3. 每个周日： 听 2 段录音，拖动五个 1–10 滑块（清晰度、流利度、发音、商务自然度、自信），写下 3 项收获 / 3 项弱点 / 3 项纠正，然后完成回顾。
  4. 第 4、8、12 周： 填写月度仪表盘——决定下一阶段的重要检查点。

11👤 账户、备份与隐私

  1. 资料： 随时编辑你的名字、身份、目标和每日时段——整个应用会重新个性化。
  2. 方案： 免费方案包含一切；高级版（AI 教练、云账户、团队仪表盘）将稍后推出。
  3. 备份： 点击 导出 JSON 每周一次并妥善保存文件； 导入 可在任何设备上恢复它。可选：连接一个私有 GitHub 仓库，自动云同步进度和录音。
  4. 隐私： 一切——进度、录音、文字稿——都保存在你的设备上。除非 你 设置了 GitHub 同步，否则不会上传任何内容。
⛔ “重置一切” 会永久清除本设备上的所有进度和录音。请先导出备份。

13常见问题与故障排除

麦克风 / 口语反馈无法使用

使用 Chrome 或 Edge ——语音识别需要它们的引擎（Safari 部分可用；Firefox 不行）。点击地址栏中的 🔒 图标 → 麦克风 → 允许 → 重新加载。在 macOS 上还请检查系统设置 → 隐私 → 麦克风。

我刷新后，片段/录音好像不见了

已修复——刷新后 Shadow 标签页现在会自动恢复你上一个片段、标记、文字稿和录音。录音始终能在刷新中保留；它们存放在你浏览器的数据库中。

我的进度消失了

你可能打开了另一个浏览器或隐私窗口。回到你常用的浏览器，或恢复你的 JSON 备份（👤 账户）。这就是每周导出习惯为何重要。

我的录音显示 0:00 / 无法分析

它太短了。请至少说 20 秒 ——你每多说一句，反馈质量就越高。（刚录好的录音在你按播放前可能显示“0:00”——这是浏览器的小毛病，并非数据丢失。）

我可以把它安装成应用吗？

可以——在 Android Chrome 上：菜单 ⋮ → 添加到主屏幕 / 安装应用。它会全屏打开，带底部导航，并可离线使用。

姿态教练会录下我吗？

不会。摄像头画面在你的设备上逐帧分析，每一帧都会立即丢弃。绝不会保存或传输任何视频或图像——教练运行期间只存在匿名的姿态坐标（数字）。关闭教练会完全释放摄像头。

我的声音会传到服务器吗？

录音保留在你的设备上。用于转写的语音引擎是你浏览器自带的（作为其语音服务的一部分，Chrome 可能会在 Google 的服务器上处理音频）；词典释义以匿名方式获取。除非你启用 GitHub 同步，否则没有其他内容会离开你的设备。

我应该按什么顺序完成这些周？

按顺序，1 → 12。各阶段层层递进，第 4、8、12 周以月度仪表盘结束。
