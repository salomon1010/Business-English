1시작하기 및 초기 설정

이 앱은 전문적인 영어 말하기를 위한 완전한 12주 훈련 프로그램입니다: 주 6일, 하루 25분 집중, 모든 녹음에 즉각적인 피드백이 제공됩니다. 모든 것은 여러분의 기기에 자동으로 저장됩니다.

  1. 앱 열기 Chrome 또는 Edge에서(권장) — https://app.lomonec.com/. Android에서는 브라우저 메뉴 → 홈 화면에 추가 를 사용해 실제 앱처럼 설치하세요(오프라인에서도 작동합니다).
  2. 60초 환영 과정을 완료하세요: 이름 → 직업 → 최우선 목표 → 매일 연습 시간대 → 기준 점수를 위해 한 문장을 소리 내어 말하기. 앱 전체가 여러분의 답변에 맞춰 개인화됩니다.
  3. 마이크 접근 허용 — 요청받으면 허용하세요. 녹음기와 말하기 피드백에 필요합니다.
  4. 🎯 Shadow 탭에 도착합니다 — 시작 영상을 골라 첫 30초 스프린트를 해보세요. 그러면 1일 차 완료입니다.
💡 황금률: 모든 세션은 반드시 증거 를 남겨야 합니다 — 녹음, 전사, 저장한 단어, 또는 교정 메모. 녹음도 기록도 하지 않았다면 진짜 세션이 아닙니다.

3대시보드

결정할 필요 없음: 대시보드가 이름으로 여러분을 맞이하고, 목표와 매일 시간대를 보여주며, 오늘의 세션이 무엇인지 정확히 알려줍니다.

  1. "이어서 계속하기" 카드를 읽고 클릭하세요 오늘의 25분 세션 시작 →.
  2. 통계를 살펴보세요: 🔥 연속 기록 · 🎙️ 녹음 · 💬 익힌 표현 /152 · ✅ 세션 /84 · 🎯 저장한 섀도잉 클립.
  3. 링은 세 단계에 걸친 84개 세션 중 완료 비율(%)을 보여줍니다: 기초(1~4주) → 직장 유창성(5~8주) → 임원급 커뮤니케이션(9~12주).

5매일 25분 세션

한 페이지에 전체 훈련이 담겨 있습니다: 타이머, 분 단위 템플릿, 실시간 전사가 되는 녹음기, 여러분의 스크립트, 메모, 자기 채점.

  1. ▶ 시작 을 눌러 25분 타이머를 켜고, 시간이 정해진 템플릿 단계를 따라가며 하나씩 체크하세요.
  2. "🎬 내 예시 — 내가 말할 내용"을 작성하세요 — 자신의 말로 된 답변(예: 자기소개). 이것이 녹음이 채점되는 기준 목표입니다.
  3. 🎙️ 를 누르고 말하세요. 녹음하는 동안 앱이 여러분의 말을 전사합니다. 목표는 20~60초 의 실제 말하기입니다 — 1초짜리 클립은 분석할 수 없습니다.
  4. 정지 → 전체 피드백 보고서가 나타납니다 (7절 참조): 단어별 점수, 문법, 어휘, 속도. 여러분의 말은 "🎙️ 내 전사"에도 자동으로 채워집니다.
  5. 두 번째 시도를 녹음하세요 — 📈 파형이 새 목소리를 이전 것 위에 겹쳐 보여주며, 다음과 같은 속도 판정이 나옵니다 "4.5초 더 짧아짐 — 더 자신 있는 속도 📈".
  6. 가장 중요한 교정을 메모에 적고, 1~10으로 자기 채점한 뒤 ✅ 세션 완료로 표시하세요.
📌 예시 — 1주 차 화요일, "자기소개를 해보세요"

내 예시:

"I currently work as a data engineer. My focus is on the reporting pipeline. I'm responsible for data quality."

녹음하면 보고서에 표시됩니다: 95% — 노란 단어 하나(responsible), 문법 지적("I am agree" → "I agree"), 속도 132 단어/분 ✅. 🔊/🎤 로 노란 단어를 연습하고 두 번째 시도를 녹음하면, 시도 줄에 다음과 같이 표시됩니다 82% → 95% 📈.

6🎯 섀도잉 스튜디오 — 연습 실험실

섀도잉 = 원어민과 함께 말하며 원어민과 함께 말하며 그들의 리듬, 강세, 억양을 따라 하는 것입니다. 스튜디오는 이를 손쉽게 만들어 줍니다: 내장 라이브러리에서 클립을 고르거나(전사 포함) 아무 YouTube 링크를 붙여넣고, 20~30초 구간을 표시한 뒤 섀도잉하며 반복 재생하세요.

클립을 잘라 반복 재생하기

  1. 영상을 고르세요: 아래 🎬 라이브러리에서 클립을 고르거나(Jobs, MLK, Michelle Obama, TED 강연 등 다수 — 각각 “자막 준비됨” 배지가 표시됩니다) 아무 YouTube 링크를 붙여넣으세요 → ▶ 영상 불러오기.
  2. 재생한 다음 ⬇ 시작 표시. 실시간 카운터가 나타나 초를 셉니다 — 20초에 녹색이 되고("이제 끝을 표시할 수 있습니다 ✅"), 30초에 금색으로 깜박입니다.
  3. ⬇ 끝 표시 → 이제 클립이 자동으로 반복 재생됩니다 🔁. 재생 중 카운터가 현재 위치를 보여줍니다: 12s / 30s, 반복할 때마다 0으로 초기화됩니다.
  4. 전사가 자동으로 채워집니다. 라이브러리 클립에서는 끝을 표시하는 순간, 앱이 바로 그 구간의 자막을 “🎬 클립 전사 및 메모” 상자로 가져오며 이미 강세가 표시되어 있습니다(대문자 = 강세 부분). 표시를 바꾼 뒤 “📄 전사 가져오기”로 다시 가져오거나 “✕ 지우기”로 새 클립을 시작하세요.
  5. 클립 안에서 이동하기 ⟲(처음부터), ↺5 / 5↻(5초 뒤로/앞으로 — 클릭이 누적됨), 속도 버튼 (1× → .75× → .5×), 🔁 반복 켜기/끄기를 사용합니다.
  6. ⭐ 클립 저장 — 저장한 클립은 한 번의 탭으로 다시 불러와지며, 표시, 전사, 녹음이 그대로 유지됩니다. 새로고침해도 마지막 클립이 자동으로 복원됩니다.

🧍 자세 코치 — 말하는 동안 임원급 존재감

  1. "🧍 자세 코치"를 클릭 (녹음기 아래, Shadow 또는 아무 세션 페이지)하고 카메라를 허용하세요.
  2. 3초 동안 바르게 앉으세요 — 코치가 여러분의 바른 자세에 맞춰 조정합니다.
  3. 평소처럼 연습하세요. 작게 빛나는 뼈대 아바타(여러분의 영상이 아님)가 여러분을 비춥니다: 녹색 = 임원급 자세 · 노란색 = 머리가 기울어짐, 휴대폰을 올리세요 · 주황색 빛 = 구부정함, 어깨를 뒤로.
  4. 어두운 방에서는 화면이 자동으로 💡 링 라이트 가 되어 얼굴을 비춥니다(한 번 탭하면 꺼짐). 추적이 여러분을 볼 수 없으면 자연스럽게 일시정지하고 목소리에 집중하게 해줍니다.
  5. 코치를 정지 하면 세션 결과가 나옵니다 — 예: "🧍 바른 자세 88%" — 조언과 함께.
🔒 무캡처 개인정보 보호: 카메라 프레임은 여러분의 기기에서 분석된 뒤 즉시 폐기됩니다. 어떤 영상이나 사진도 저장되거나 업로드되지 않습니다 — 익명의 수치 자세 좌표만 존재하며, 그것들은 결코 기기를 벗어나지 않습니다.

섀도잉하고 녹음하기

  1. 반복 구간을 6~8회 섀도잉: 듣기 → 빈 곳에서 따라 하기 → 오디오와 함께 말하기 → 혼자 말하기.
  2. 🎬 클립 전사를 확인하세요. 라이브러리 클립은 표시한 구간에서 이미 채워지고 강세가 표시되어 있습니다. 붙여넣은 영상은 화자가 하는 말을 입력하고 “✨ 강세 표시”를 누르세요. 강세 부분은 대문자로 표시되고 쉼은 /로 표시됩니다.
  3. 🎙️ 를 누르고 한 번을 통째로 섀도잉하세요. 정지 시: 여러분의 말이 🎙️ 내 전사에 나타나며, 바로 그 녹음으로부터 전체 말하기 피드백 보고서가 생성됩니다.
  4. 클립 반복하기 — 매 시도가 하나의 도전입니다: "이 클립 · 시도 3 — 80% → 95% → 100% 📈 향상되고 있어요!"

7🗣️ 말하기 피드백 설명

녹음할 때마다(또는 🎤 말하기 피드백 시작 으로 실시간 "소리 내어 읽기" 후에), 앱은 여러분이 실제로 말한 내용으로 전체 보고서를 만듭니다.

항목무엇을 알려주는가
점수 %미국식 영어 음성 엔진이 여러분의 목표 단어를 얼마나 명확하게 인식했는지
단어 칩녹색 = 명확히 전달됨 · 노란색 = 연습 필요(마우스를 올리면 대신 무엇으로 들렸는지 표시)
🎯 연습 카드문제가 되는 각 단어에 대해: 🔈 당신 (여러분 자신의 녹음에서 그 순간을 재생) · 🔊 예시 · 🐢 느리게 · 🎤 말해보기 — 다시 시도 (정확히 해내면 100%에서 녹색으로 변함)
속도와 군말여러분의 분당 단어 수와 원어민의 140~160 구간 비교, 그리고 "um/uh" 횟수
시도이 클립/세션의 점수 기록과 📈/📉 판정
📖 문법 피드백두 개의 열 — 잘한 점 (조동사, present perfect, 표현 모음의 표현 사용!) 그리고 개선할 수 있는 점 ("informations" → information, "I am agree" → I agree…), 각각에 🔊 올바른 버전 듣기 포함
📚 핵심 어휘여러분의 말에서 추출된 단어와 CEFR 등급 — 8절 참조
💡 20초 이상 말하세요. 많이 말할수록 보고서가 풍부해집니다. 그리고 이것은 근삿값임을 기억하세요: "놓친" 단어는 대개 불명확한 발음을 뜻하지만, 배경 소음도 영향을 줍니다.

8📚 어휘, 어려운 단어, 퀴즈

  1. 핵심 어휘 는 모든 피드백 보고서에 나타납니다 — 여러분 자신의 말 에서 나온 단어이며 등급 배지가 붙습니다(A2…C1). 단어를 탭하면 그 사전 정의, IPA, 실제 사람 음성이 나옵니다; 🎤 말해보기 로 스스로 테스트하고; 🔖 저장 해 좋은 것을 남기세요(또는 "모든 단어 저장").
  2. ⬆️ 단어 수준 높이기: 앱이 여러분이 사용한 캐주얼한 단어를 찾아 임원급 표현으로 업그레이드를 제안합니다 — 당신은 "get"이라고 말했습니다 → "obtain"을 시도해 보세요 — 음성과 저장 버튼과 함께.
  3. 💪 내 어려운 단어 (Shadow 하단): 피드백이 표시한 모든 단어가 여기에 자동으로 모입니다. 🔊 / 🐢 / 🎤 로 각각 연습하세요 — 틀린 횟수만큼 정확히 발음하면 🏆 와 함께 지워집니다.
  4. 📚 내 어휘 + 🎴 퀴즈 내기: 저장한 단어가 덱이 됩니다. 퀴즈는 단어의 뜻 을 읽어주고 — 여러분은 그 단어를 소리 내어 말해야 합니다; 앱은 기억과 발음을 모두 판정합니다.
📌 예시 — 퀴즈

"이 단어는 무엇일까요? — The people with an interest or concern in a business project."

여러분은 🎤 를 누르고 말합니다 "stakeholders" → "🏆 정답이면서 발음도 좋아요!"

9표현과 💼 임원급 다듬기

💼 임원급 다듬기 — 리더처럼 말하기

  1. 입력하거나 🎤 받아쓰기 로 캐주얼하게 할 말을 넣고 누르세요 ✨ 다듬기.
  2. 임원급 버전을 읽으세요 — 군말 제거, 얼버무리는 표현 삭제("I think", "maybe", "sorry about that"), 문법 수정, 약한 단어 업그레이드 — 모든 변경 사항이 나열되어 그 패턴을 배울 수 있습니다.
  3. 🔊 듣기, 🐢 천천히, 📋 복사 — 또는 누르세요 🤖 AI 코치 프롬프트 를 눌러 ChatGPT, Claude 또는 Gemini용으로 (여러분의 문장이 들어간) 완성된 코칭 프롬프트를 복사하세요.
📌 예시

"um I think maybe we need to fix the problem with the informations"

→ "I recommend we resolve the challenge with the information."

표현 모음

주별 주제에 맞춘 10~13개의 표현과 관용구. 각각: 5회 말하고, 🎙️ 녹음 해 자신의 문장을 남기고, 익힘으로 표시 자연스럽게 자신의 것이 되면 표시하세요. 일요일 할 일: 다음 주 실제 회의에서 쓸 5개를 고르세요.

10리뷰와 📈 진행 스튜디오

  1. 📖 이번 주 나의 이야기 — 여러분 연습의 자동 작성 요약: 일수, 말한 단어 수, 점수 추세, 새 어휘, 지워야 할 어려운 단어.
  2. 네 개의 차트 — 🎯 발음 점수 여정 · 📚 어휘 성장 · ⚡ 말하기 속도 게이지(워밍업 / 딱 좋음 / 빠름) · 🗓️ 하루에 말한 단어 수. 모두 자동으로 측정됩니다.
  3. 매주 일요일: 녹음 2개를 듣고, 다섯 개의 1~10 슬라이더(명료성, 유창성, 발음, 비즈니스 자연스러움, 자신감)를 움직이고, 3가지 성과 / 3가지 약점 / 3가지 교정을 적은 뒤 리뷰를 완료하세요.
  4. 4·8·12주: 월간 대시보드를 작성하세요 — 다음 단계를 정하는 큰 점검 지점입니다.

11👤 계정, 백업, 개인정보

  1. 프로필: 이름, 역할, 목표, 매일 시간대를 언제든 수정하세요 — 앱 전체가 다시 개인화됩니다.
  2. 요금제: 무료 요금제에 모든 것이 포함됩니다; 프리미엄(AI 코치, 클라우드 계정, 팀 대시보드)은 추후 제공됩니다.
  3. 백업: 클릭 JSON 내보내기 를 매주 하고 파일을 안전하게 보관하세요; 가져오기 는 어떤 기기에서든 복원합니다. 선택 사항: 비공개 GitHub 저장소를 연결하면 진행 상황과 녹음이 자동으로 클라우드 동기화됩니다.
  4. 개인정보: 모든 것 — 진행 상황, 녹음, 전사 — 은 여러분의 기기에 있습니다. 다음의 경우가 아니면 아무것도 업로드되지 않습니다: 여러분 이 GitHub 동기화를 설정한 경우.
⛔ "모두 초기화" 는 이 기기의 모든 진행 상황과 녹음을 영구적으로 지웁니다. 먼저 백업을 내보내세요.

13자주 묻는 질문 및 문제 해결

마이크 / 말하기 피드백이 작동하지 않아요

사용하세요 Chrome 또는 Edge 를 — 음성 인식에는 이들의 엔진이 필요합니다(Safari는 부분 작동, Firefox는 미지원). 주소창의 🔒 아이콘 클릭 → 마이크 → 허용 → 새로고침. macOS에서는 시스템 설정 → 개인정보 보호 → 마이크 도 확인하세요.

새로고침했더니 클립/녹음이 사라진 것 같아요

해결됨 — 이제 Shadow 탭은 새로고침 후 마지막 클립, 표시, 전사, 녹음을 자동으로 복원합니다. 녹음은 새로고침에도 항상 유지되며, 브라우저의 데이터베이스에 저장됩니다.

진행 상황이 사라졌어요

아마 다른 브라우저나 시크릿 창을 열었을 것입니다. 평소 쓰던 브라우저로 돌아가거나 JSON 백업을 복원하세요(👤 계정). 그래서 매주 내보내는 습관이 중요합니다.

녹음이 0:00으로 표시돼요 / 분석되지 않아요

너무 짧았습니다. 최소한 20초 이상 말하세요 — 한 문장 말할 때마다 피드백 품질이 높아집니다. (갓 만든 녹음은 재생을 누르기 전까지 "0:00"으로 표시될 수 있습니다 — 이는 브라우저의 특성이며 데이터 손실이 아닙니다.)

앱으로 설치할 수 있나요?

네 — Android Chrome에서: 메뉴 ⋮ → 홈 화면에 추가 / 앱 설치. 하단 탐색이 있는 전체 화면으로 열리며 오프라인에서도 작동합니다.

자세 코치가 저를 녹화하나요?

아니요. 카메라 영상은 여러분의 기기에서 프레임 단위로 분석되고 각 프레임은 즉시 폐기됩니다. 어떤 영상이나 이미지도 저장되거나 전송되지 않습니다 — 코치가 실행되는 동안 익명의 자세 좌표(숫자)만 존재합니다. 코치를 끄면 카메라가 완전히 해제됩니다.

제 목소리가 서버로 전송되나요?

녹음은 여러분의 기기에 남습니다. 전사에 사용되는 음성 엔진은 브라우저 자체의 것입니다(Chrome은 음성 서비스의 일부로 Google 서버에서 오디오를 처리할 수 있습니다); 사전 정의는 익명으로 가져옵니다. GitHub 동기화를 켜지 않는 한 그 밖에 기기를 벗어나는 것은 없습니다.

주는 어떤 순서로 해야 하나요?

순서대로, 1 → 12입니다. 각 단계는 서로 쌓여가며, 4·8·12주는 월간 대시보드로 마무리됩니다.
